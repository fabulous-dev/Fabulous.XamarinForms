## Basic example

```fs
View.ProgressBar(progress = 0.5)
```

## Basic example with styling

```fs
View.ProgressBar(
    horizontalOptions = style.Position,
    verticalOptions = style.Position,
    backgroundColor = style.LayoutColor,
    progress = 0.5
)
```
